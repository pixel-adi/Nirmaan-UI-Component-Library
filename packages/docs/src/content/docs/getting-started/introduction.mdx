**Nirmaan UI** is an elite, multi-platform Design System built to bridge the gap between design tokens and engineering execution.

## The Problem Nirmaan Solves
Most design systems build isolated components for React, leaving iOS and Android engineers to manually recreate Figma specifications. This leads to drift—colors mismatches, incorrect touch targets, and fractured User Experiences.

## Our Philosophy
1. **Design Tokens are the Source of Truth:** Everything from spacing units (based on a `4px` grid) to typography weights are defined in raw JSON primitives.
2. **Universal Compilation:** Those JSON tokens automatically transform into native `Swift` structs for iOS, native `Kotlin` classes for Compose, and native `CSS` variables for web apps.
3. **Accessibility by Default:** Our components are hard-wired to pass WCAG 2.5.8 standards. If an engineer uses a `Button`, the system mathematically guarantees a 44px minimum touch target without the engineer needing to calculate it.
4. **Interactive Validation:** This documentation doesn't just display code; it natively renders the actual React components in live Playgrounds. What you see here is exactly what your web platform renders.

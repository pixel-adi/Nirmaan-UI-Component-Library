Nirmaan is a **design token-driven system** for Indian public infrastructure — built for multi-platform delivery across Web, iOS, and Android. It provides a single source of truth defined in Figma, transformed by Style Dictionary, and consumed by any platform.

> [!NOTE]
> **One design decision. Every platform.**
> `Figma Variables → Style Dictionary → CSS / Swift / Android XML / SCSS / JS`

## Token Naming Convention

All tokens follow the path structure from Figma: `collection / group / subgroup / variant`

| Platform | Format | Example |
| :--- | :--- | :--- |
| **CSS** | `--nir-{path}` | `--nir-colour-primary-base` |
| **SCSS** | `$nir-{path}` | `$nir-colour-primary-base` |
| **Swift** | `Nir.Colour.Primary.base` | `UIColor(hex: "#083459")` |
| **Android** | `nir_colour_primary_base` | `@color/nir_colour_primary_base` |
| **JS/TS** | `NirColourPrimaryBase` | `"#083459"` |

## The Six Design Principles

<CardGrid stagger>
  <Card title="1. Token-First" icon="approve-check-circle">
    No hardcoded values. Every colour, spacing, and radius is a design token. Changing one value propagates across the entire system and every platform instantly.
  </Card>
  <Card title="2. Multi-Platform" icon="laptop">
    One definition serves Web, iOS, Android, and beyond. Style Dictionary handles the transformation so your team never defines a value twice.
  </Card>
  <Card title="3. Janabhasha (जनभाषा) — Multi-Script" icon="document">
    Built for 12 Indian scripts with per-script typographic metrics. Shirorekha-aware line heights are **mathematically enforced** in the token architecture.
  </Card>
  <Card title="4. Sarvajanik (सार्वजनिक) — Accessible" icon="magnifier">
    WCAG AA minimum, enforced in the token layer. High-contrast mode and 44×44px touch targets are structurally clamped — not a checklist item.
  </Card>
  <Card title="5. Vishvasaniya (विश्वसनीय) — Open" icon="information">
    All tools are open-source. JSON is the source of truth. No vendor lock-in. Your tokens are portable to any future system.
  </Card>
  <Card title="6. Surakshit (सुरक्षित) — Trustworthy" icon="error">
    Strict API contracts between Figma and Engineering. Interaction patterns like OTP masking and session timeouts are secure by default.
  </Card>
</CardGrid>

## For Developers

<Steps>

1. **Install the token package**

   ```bash
   npm install @nirman/tokens
   ```

2. **Import for your platform**

   <Tabs>
     <TabItem label="Web (CSS)">
       Import at your app entry point (`_app.tsx`, `index.ts`, or `main.ts`):
       ```tsx
       import '@nirman/tokens/css/variables.css';
       ```
       All tokens become CSS custom properties:
       ```css
       .my-button {
         background: var(--nir-colour-primary-base);
         color: var(--nir-colour-primary-on-base);
         padding: var(--nir-spacing-vertical-base) var(--nir-spacing-horizontal-base);
         border-radius: var(--nir-spacing-corner-radius-base);
       }
       ```
     </TabItem>
     <TabItem label="Web (SCSS)">
       ```scss
       @import '@nirman/tokens/scss/variables';

       .my-button {
         background: $nir-colour-primary-base;
         border-radius: $nir-spacing-corner-radius-base;
       }
       ```
     </TabItem>
     <TabItem label="iOS (Swift)">
       ```swift
       import NirTokens

       let button = UIButton()
       button.backgroundColor = NirColour.primaryBase
       button.layer.cornerRadius = NirSpacing.cornerRadiusBase
       button.contentEdgeInsets = UIEdgeInsets(
         top: NirSpacing.verticalBase,
         left: NirSpacing.horizontalBase,
         bottom: NirSpacing.verticalBase,
         right: NirSpacing.horizontalBase
       )
       ```
     </TabItem>
     <TabItem label="Android (XML)">
       ```xml
       <Button
         android:background="@color/nir_colour_primary_base"
         android:textColor="@color/nir_colour_primary_on_base"
         android:paddingHorizontal="@dimen/nir_spacing_horizontal_base"
         android:paddingVertical="@dimen/nir_spacing_vertical_base"
         app:cornerRadius="@dimen/nir_spacing_corner_radius_base" />
       ```
     </TabItem>
     <TabItem label="JavaScript / TypeScript">
       ```tsx
       import { NirColourPrimaryBase, NirSpacingCornerRadiusBase } from '@nirman/tokens/js';

       const styles = {
         backgroundColor: NirColourPrimaryBase,
         borderRadius: `${NirSpacingCornerRadiusBase}px`,
       };
       ```
     </TabItem>
   </Tabs>

3. **Build tokens locally (for contributors)**

   ```bash
   git clone https://github.com/your-org/nirman.git
   cd nirman/packages/tokens
   pnpm install
   node sd.config.mjs
   ```

   Output appears in the `build/` folder — one subfolder per platform.

</Steps>

## For Designers

<Steps>

1. **Enable the Figma library**
   1. Open any Figma file
   2. Click **Assets** panel (left sidebar)
   3. Click the **book icon** (Team Library)
   4. Enable **Nirman Design System** library

2. **Use variables**

   All design values come from the Variables panel:
   1. Select an element
   2. Click fill, stroke, padding, or radius property
   3. Click the **variable icon** (hexagon)
   4. Browse `global-variables` for the token you need

3. **Key rules**
   - Always use variables from `global-variables`
   - Check that Dev Mode shows `--nir-*` names
   - Test your design in multiple scripts (Hindi, Tamil)
   - **Never** hardcode hex colours or pixel values
   - **Never** use `tokens` collection directly (those are primitives)
   - **Never** detach a component instance

4. **Exporting tokens (Design Lead only)**

   When variables change:
   1. Open Figma → Plugins → run export plugin
   2. Save `export.json` to `packages/tokens/`
   3. Run `node sd.config.mjs` to rebuild all platforms
   4. Commit and push

</Steps>

Buttons are the primary mechanism for triggering actions. The Nirmaan Button maps strictly to Constitutional accessibility standards — every sizing, colour, and interaction state is token-bound.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Role** | `button` (native `<button>` element) | Never use `<div>` or `<span>` as a button. |
| **Label** | Explicit text content or `aria-label` | Screen readers announce the label as "[text], button". |
| **Keyboard** | `Enter` and `Space` to activate | Native to `<button>` — no custom JS required. |
| **Disabled** | `aria-disabled="true"` | Keeps the element in tab order so users know it exists. |
| **Loading** | `aria-busy="true"` | Announces "busy" to screen readers during loading state. |
| **Focus ring** | 2px solid, 2px offset | Uses `--nir-colour-primary-base` against page background. |
| **Touch target** | Minimum 44×44px | Enforced via `min-height: var(--nir-spacing-touch-target)`. |

## 4. Component Anatomy

```
┌─────────────────────────────────┐
│  [icon?]  Label Text  [icon?]   │  ← Button container
└─────────────────────────────────┘
     │           │
     │           └── .nir-e-button__label  → font: label-md (medium, 16px)
     └──────────── .nir-e-button__icon  → size: 20×20px
```

## 5. Variants & Sizes

| Variant | Use case | Fill | Border |
| :--- | :--- | :--- | :--- |
| `primary` | Main CTA, submit actions | `--nir-colour-primary-base` | None |
| `secondary` | Cancel, back, secondary actions | `--nir-colour-secondary-base` | `--nir-colour-primary-stroke` |
| `danger` | Destructive actions (delete, revoke) | `--nir-colour-status-negative-base` | None |
| `ghost` | Low-emphasis, inline actions | Transparent | None |

| Size | Height | Padding H | Font size |
| :--- | :--- | :--- | :--- |
| `sm` | 36px | 12px | 14px |
| `md` (default) | 44px | 16px | 16px |
| `lg` | 52px | 24px | 18px |

            .foregroundColor(Color(NirColourPrimaryOnBase))

    .frame(minHeight: NirSpacingTouchTarget)
    .padding(.horizontal, NirSpacingHorizontalBase)

Dropdowns govern dense contextual configuration sets bound deeply into the `--nir-elevation-overlay` logic safely avoiding canvas clipping.

## 1. Accessibility Specification
- **ARIA:** Translates actively to `<button aria-haspopup="listbox">` internally. Focus drops rapidly into the interior wrapper securely locking Tab rotations.
- **Keyboard:** Closes rigidly on global `ESC` injections.

## 2. Token Binding
- **Elevation Layer:** The Z-Bounds strictly bind to `elevation.dropdown` and drop box shadows per `elevation.3`.
- **Text Bounds:** Selectable matrices parse directly over array borders utilizing exactly `font.size.body-md`.

Dropdowns present a list of options from which the user selects one value. The list is hidden until triggered, keeping the interface clean. The overlay panel renders above other content using `--nir-z-index-dropdown`.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Role** | `combobox` on trigger | `<button role="combobox" aria-haspopup="listbox">` |
| **List role** | `listbox` on options panel | `<ul role="listbox">` |
| **Option role** | `option` per item | `<li role="option" aria-selected="true/false">` |
| **Open state** | `aria-expanded="true/false"` | Toggled on open/close. |
| **Active item** | `aria-activedescendant` | Points to the currently highlighted option ID. |
| **Keyboard** | `Enter`/`Space` opens, `↑↓` navigate, `Enter` selects, `Esc` closes | Full keyboard control required. |
| **Label** | Visible `<label>` linked to trigger | Always present — placeholder alone is not sufficient. |
| **Touch target** | Minimum 44px height on trigger | The closed button state must meet touch target. |

## 4. Component Anatomy

```
┌─ label ──────────────────────────────────┐
│ Select State                             │
├──────────────────────────────────────────┤
│  Selected value or placeholder      [∨]  │  ← trigger
└──────────────────────────────────────────┘
  ┌──────────────────────────────────────┐
  │  Option A                            │  ← panel (elevation-4)
  │  Option B  ✓ (selected)              │
  │  Option C                            │
  └──────────────────────────────────────┘
```

  <TabItem label="SwiftUI (iOS)">
    ```swift
    import NirmaanTokens

    Menu {
        ForEach(states) { state in
            Button(state.name) { selectedState = state }
        }
    } label: {
        HStack {
            Text(selectedState?.name ?? "Select State")
                .foregroundColor(
                    selectedState == nil
                        ? Color(NirColourTextPlaceholder)
                        : Color(NirColourTextDefault)
                )
            Spacer()
            Image(systemName: "chevron.down")
                .foregroundColor(Color(NirColourIconDefault))
        }
        .padding(.horizontal, NirSpacingHorizontalBase)
        .padding(.vertical, NirSpacingVerticalSmaller)
        .frame(minHeight: NirSpacingTouchTarget)
        .background(Color(NirColourSurfaceContainer))
        .cornerRadius(NirSpacingCornerRadiusBase)
        .overlay(
            RoundedRectangle(cornerRadius: NirSpacingCornerRadiusBase)
                .stroke(Color(NirColourBorderDefault), lineWidth: 2)
        )
    }
    .accessibilityLabel("State selector")
    ```
  </TabItem>
  <TabItem label="Compose (Android)">
    ```kotlin
    import dev.nirmaan.tokens.NirmaanTheme

    ExposedDropdownMenuBox(
        expanded = expanded,
        onExpandedChange = { expanded = !expanded }
    ) {
        OutlinedTextField(
            value = selectedOption,
            onValueChange = {},
            readOnly = true,
            label = { Text("Select State") },
            trailingIcon = {
                ExposedDropdownMenuDefaults.TrailingIcon(expanded = expanded)
            },
            colors = OutlinedTextFieldDefaults.colors(
                focusedBorderColor = NirmaanTheme.colors.borderFocus,
                unfocusedBorderColor = NirmaanTheme.colors.borderDefault,
            ),
            modifier = Modifier
                .menuAnchor()
                .fillMaxWidth()
                .defaultMinSize(minHeight = NirmaanTheme.spacing.touchTarget)
        )
        ExposedDropdownMenu(expanded = expanded, onDismissRequest = { expanded = false }) {
            states.forEach { state ->
                DropdownMenuItem(
                    text = { Text(state.name) },
                    onClick = { selectedOption = state.name; expanded = false }
                )
            }
        }
    }
    ```

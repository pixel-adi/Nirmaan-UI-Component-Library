Text inputs govern standard data entry schemas on Web and Native apps natively bounded to Constitutional definitions. 

## 1. Accessibility Specification
- **ARIA:** Natively constructs IDs automatically linking explicit standard `aria-describedby` when errors map to inputs.
- **Focus Indicators:** Binds cleanly to `--nir-interaction-focus-ring` (minimum 3:1 AAA ratio against canvas bounding).
- **Min Touch Bounds:** The interactive container hits a `44px` hard minimum.

## 2. Token Binding
- **Border:** `color.border.default` mapping properly to `color.border.focus` (Interaction semantic).
- **State Feedback:** Error configurations instantly pivot to `color.feedback.error.border` without manual injection.

    TextField("Aadhaar Hash", text: $val)
      .padding(Tokens.Spacing.componentPaddingMd)
      .overlay(
          RoundedRectangle(cornerRadius: Tokens.Radius.interactive)
              .stroke(Tokens.Color.Feedback.Error.border, lineWidth: 2)
      )
      .frame(minHeight: Tokens.Spacing.touchTarget) 
    ```

    )
    ```

Text inputs govern standard data entry. Every border, label, error colour, and spacing value is token-driven — no hardcoded values exist in the component implementation.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Label** | Always visible `<label>` linked via `htmlFor`/`id` | Floating or fixed — never placeholder-only labels. |
| **Error** | `aria-invalid="true"` + `aria-describedby` pointing to error text | Screen readers announce "invalid entry" automatically. |
| **Error message** | `role="alert"` | Triggers immediate announcement on error state change. |
| **Helper text** | `aria-describedby` | Linked to input — announced after label. |
| **Required** | `aria-required="true"` | Announced as "required field". |
| **Focus ring** | 2px solid `--nir-colour-border-focus`, 0px offset (inside) | Inset outline to avoid layout shift. |
| **Touch target** | Minimum 44px height | Enforced via `min-height: var(--nir-spacing-touch-target)`. |
| **Disabled** | `aria-disabled="true"` + `cursor: not-allowed` | Not HTML `disabled` — keeps tab navigation intact. |

## 4. Component Anatomy

```
┌─ label ─────────────────────────────┐
│ Full Name                           │
├─────────────────────────────────────┤
│  Input text or placeholder          │  ← .nir-e-input__field
│                                     │
└─────────────────────────────────────┘
  Helper text or error message          ← .nir-e-input__helper / __error
```

## 5. States Reference

| State | Border token | Background | Description |
| :--- | :--- | :--- | :--- |
| Default | `--nir-colour-border-default` | `--nir-colour-surface-container` | Resting state |
| Focused | `--nir-colour-border-focus` | `--nir-colour-surface-container` | 2px navy inset ring |
| Error | `--nir-colour-border-error` | `--nir-colour-feedback-error-subtle` | Invalid data |
| Success | `--nir-colour-border-success` | `--nir-colour-surface-container` | Validated |
| Disabled | `--nir-colour-border-disabled` | `--nir-colour-interaction-disabled-bg` | 0.38 opacity |

    import { Input } from '@nirman/components';

    // All ARIA links, token states, and announcements are automatic
    <Input
      label="Aadhaar ID"
      placeholder="Enter 12-digit number"
      helperText="Find your Aadhaar on your e-Aadhaar card"
      required
      fullWidth

      fullWidth
    />

  <TabItem label="SwiftUI (iOS)">
    ```swift
    import NirmaanTokens

    VStack(alignment: .leading, spacing: NirSpacingVerticalSmallest) {
        Text("Aadhaar ID")
            .font(.custom(NirFontFamilyBody, size: NirFontSizeBodySm))
            .foregroundColor(Color(NirColourTextSecondary))

        TextField("Enter 12-digit number", text: $value)
            .padding(.horizontal, NirSpacingHorizontalBase)
            .padding(.vertical, NirSpacingVerticalSmaller)
            .frame(minHeight: NirSpacingTouchTarget)
            .overlay(
                RoundedRectangle(cornerRadius: NirSpacingCornerRadiusBase)
                    .stroke(
                        hasError ? Color(NirColourBorderError) : Color(NirColourBorderDefault),
                        lineWidth: 2
                    )
            )
            .accessibilityLabel("Aadhaar ID")
            .accessibilityHint(hasError ? "Invalid format" : "Enter 12-digit number")
    }
    ```
  </TabItem>
  <TabItem label="Compose (Android)">
    ```kotlin
    import dev.nirmaan.tokens.NirmaanTheme

    OutlinedTextField(
        value = text,
        onValueChange = { text = it },
        label = { Text("Aadhaar ID") },
        isError = hasError,
        supportingText = {
            if (hasError) Text("Invalid format", color = NirmaanTheme.colors.feedbackErrorText)
        },
        colors = OutlinedTextFieldDefaults.colors(
            focusedBorderColor = NirmaanTheme.colors.borderFocus,
            unfocusedBorderColor = NirmaanTheme.colors.borderDefault,
            errorBorderColor = NirmaanTheme.colors.borderError,
        ),
        modifier = Modifier
            .fillMaxWidth()
            .defaultMinSize(minHeight = NirmaanTheme.spacing.touchTarget)
    )
    ```

Toggles initiate instant logic modifications without demanding a structural "Save" button click afterwards.

## 1. Accessibility Specification
- **ARIA:** Formally translates directly to `<button role="switch" aria-checked="true">`. 
- **Verbal Caching:** "On" and "Off" strings are formally bound to localizations, meaning SR devices explicitly define the current bounds based on screen state context.

## 2. Token Binding
- **Track Boundary:** Relates mathematically to `color.primary.default` when active and `color.interaction.disabled-bg` when disabled dynamically.
- **Node Fill:** Fixed independently to `color.surface.default`.

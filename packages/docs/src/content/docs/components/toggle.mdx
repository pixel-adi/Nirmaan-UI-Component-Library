Toggles initiate instant, reversible binary changes — no "Save" action required. They represent the current system state and should only be used when the action takes immediate effect.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Role** | `switch` | `<button role="switch">` — not `<input type="checkbox">`. |
| **Checked state** | `aria-checked="true/false"` | Announces "on" or "off" to screen readers. |
| **Label** | Always visible, adjacent to toggle | Describes what the toggle controls. |
| **Keyboard** | `Space` to toggle | Tab to focus the switch, Space to flip state. |
| **Touch target** | Minimum 44×44px | Track + padding must reach 44×44px combined. |
| **State announcement** | Uses locale-mapped "On"/"Off" strings | Allows Hindi "चालू"/"बंद" localization. |
| **Disabled** | `aria-disabled="true"` | Toggle visible but inoperable — user understands the constraint. |

## 4. Component Anatomy

```
Label text  [ ●     ]   ← OFF state
Label text  [     ● ]   ← ON state (primary fill)

  [       ] = .nir-e-toggle__track  → 40×24px visual track
       ●    = .nir-e-toggle__thumb  → 20×20px white knob
```

The outer wrapper `.nir-e-toggle` expands to 44px height via padding.

## 5. When to Use Toggle vs Checkbox

| Scenario | Use |
| :--- | :--- |
| Setting takes immediate effect (e.g., dark mode) | **Toggle** |
| Setting saved on form submit | **Checkbox** |
| Multiple independent options in a list | **Checkbox** |
| Single on/off system preference | **Toggle** |

    ```tsx
    import { Toggle } from '@nirman/components';

    // State change happens immediately — no submit button needed
    <Toggle
      label="Enable Two-Factor Authentication"
      defaultChecked={user.twoFactorEnabled}
      onChange={(isOn) => updateSetting('2fa', isOn)}
    />

    // Disabled toggle — shows state but prevents change
    <Toggle
      label="High-Contrast Mode"
      disabled
      helperText="Contact your administrator to change this setting"
    />
    ```

    ```swift
    import NirmaanTokens

    Toggle(isOn: $isActive) {
        VStack(alignment: .leading, spacing: 2) {
            Text("Enable GPS Monitoring")
                .font(.custom(NirFontFamilyBody, size: NirFontSizeBodyMd))
                .foregroundColor(Color(NirColourTextDefault))

            Text("Required for field verification")
                .font(.custom(NirFontFamilyBody, size: NirFontSizeBodySm))
                .foregroundColor(Color(NirColourTextSecondary))
        }
    }
    // tint controls the ON state track colour
    .tint(Color(NirColourPrimaryDefault))
    .frame(minHeight: NirSpacingTouchTarget)
    .accessibilityLabel("GPS Monitoring, \(isActive ? "on" : "off")")
    ```
  </TabItem>

    ```kotlin
    Row(
    ```

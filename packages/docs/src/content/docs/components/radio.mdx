Radio buttons enforce that only one option within a group can be selected at a time. They must always exist as part of a named group — a lone radio button is an accessibility violation.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Group wrapper** | `<fieldset>` + `<legend>` | Legend announces the group question to SR users. |
| **Role** | `radio` (native `<input type="radio">`) | Native element required — never custom `<div>`. |
| **Group role** | `role="radiogroup"` | Applied automatically to the wrapper. |
| **Keyboard** | `Arrow keys` to navigate within group | `Tab` moves to/from the group as a unit. |
| **Checked state** | `aria-checked="true/false"` | Native to `<input type="radio">`. |
| **Touch target** | 44×44px minimum via padding | Visual dot is 20×20px; padding extends tap area. |
| **Disabled** | `aria-disabled="true"` | Individual options can be disabled while group remains navigable. |

## 4. Component Anatomy

```
◉  Label text       ← Selected state
○  Another option   ← Unselected state
○  Disabled         ← Disabled state (0.38 opacity)

  ◉ = .nir-e-radio__dot (filled when selected)
  ○ = .nir-e-radio__ring (outer border)
```

  <TabItem label="SwiftUI (iOS)">
    ```swift
    import NirmaanTokens

    // SwiftUI doesn't have a native Radio — implement with custom toggle style
    ForEach(options) { option in
        Button(action: { selectedOption = option.id }) {
            HStack(spacing: NirSpacingHorizontalSmaller) {
                ZStack {
                    Circle()
                        .stroke(
                            selectedOption == option.id
                                ? Color(NirColourPrimaryDefault)
                                : Color(NirColourBorderStrong),
                            lineWidth: 2
                        )
                        .frame(width: 20, height: 20)

                    if selectedOption == option.id {
                        Circle()
                            .fill(Color(NirColourPrimaryDefault))
                            .frame(width: 10, height: 10)
                    }
                }
                Text(option.label)
                    .foregroundColor(Color(NirColourTextDefault))
            }
        }
        .frame(minHeight: NirSpacingTouchTarget)
        .buttonStyle(.plain)
        .accessibilityAddTraits(selectedOption == option.id ? .isSelected : [])
    }
    ```
  </TabItem>
  <TabItem label="Compose (Android)">
    ```kotlin
    import dev.nirmaan.tokens.NirmaanTheme

    options.forEach { option ->
        Row(
            verticalAlignment = Alignment.CenterVertically,
            modifier = Modifier
                .fillMaxWidth()
                .defaultMinSize(minHeight = NirmaanTheme.spacing.touchTarget)
                .selectable(
                    selected = (selectedOption == option.value),
                    onClick = { selectedOption = option.value },
                    role = Role.RadioButton
                )
        ) {
            RadioButton(
                selected = (selectedOption == option.value),
                onClick = null,
                colors = RadioButtonDefaults.colors(
                    selectedColor = NirmaanTheme.colors.primaryDefault,
                    unselectedColor = NirmaanTheme.colors.borderStrong
                )
            )
            Spacer(Modifier.width(NirmaanTheme.spacing.horizontalSmaller))
            Text(option.label, color = NirmaanTheme.colors.textDefault)
        }
    }
    ```

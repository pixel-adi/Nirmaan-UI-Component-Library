Radio buttons dictate that only a singular choice may be parsed simultaneously within a specific structure array.

## 1. Accessibility Specification
- **ARIA:** Groups must inherently be wrapped in generic `fieldset` parameters with a visually hidden `legend` outlining the group context natively.
- **Keyboard:** Explicitly utilizes arrow-keys to rotate selections natively without firing standard tab navigation recursively.

## 2. Token Binding
- **Active State:** Overrides inherently to `color.primary.default` and enforces an internal `size / radius` manipulation drawing the inner dot.
- **Touch Limits:** Bounding padding extends exactly equal to Checkbox to capture `44px` bounds securely.

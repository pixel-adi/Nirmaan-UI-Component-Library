Autocomplete interfaces break the mold. They are formally **Layer 2 (Patterns)** because they intrinsically merge a `Layer 1 Input` and a `Layer 1 Dropdown` logically.

## 1. Accessibility Specification
- **ARIA:** Maps securely wrapping standard input strings with `role="combobox"` explicitly evaluating search subsets dynamically.
- **Screen Reader Announcements:** Crucially evaluates and utilizes `aria-live="polite"` bounding box updates to instantly notify blind respondents how many elements exist!

## 2. Token Binding
- **Nested Tiers:** Composes the semantic tokens perfectly—border colors explicitly translate to `--nir-border-focus` while simultaneously spinning up depth mapping via `--nir-elevation-overlay`.

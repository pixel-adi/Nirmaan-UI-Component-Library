Autocomplete is a **Layer 2 (Pattern)** component — it composes a `Layer 1 Input` and a `Layer 1 Dropdown` into a unified predictive search experience. As the user types, the option list filters in real time and results are announced to screen readers via `aria-live`.

## 6. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Role** | `combobox` on input | `<input role="combobox" aria-autocomplete="list">` |
| **List role** | `listbox` on suggestions panel | `<ul role="listbox">` |
| **Option role** | `option` per suggestion | `<li role="option" aria-selected="true/false">` |
| **Active item** | `aria-activedescendant` | Updated to the currently highlighted option ID. |
| **Open state** | `aria-expanded="true/false"` | True when suggestions panel is visible. |
| **Keyboard** | Type to filter, `↑↓` navigate, `Enter` select, `Esc` close | Full keyboard navigation without mouse. |
| **Live region** | `aria-live="polite"` on results count | Announces "3 results found" after each keystroke. |
| **Touch target** | 44px input height minimum | Standard input sizing applies. |

## 8. Engineering Implementation

<Tabs>
  <TabItem label="React">
    ```tsx
    import { Autocomplete } from '@nirman/components';

    // Uncontrolled — component manages its own input state
    <Autocomplete
      label="State"
      placeholder="Type to search…"
      variant="outlined"   // "outlined" | "filled" | "underline"
      size="md"            // "sm" | "md" | "lg"
      options={states.map(s => ({ label: s.name, value: s.code }))}
      helperText="Select the state where the incident occurred"
      fullWidth
    />

    // Controlled — parent owns the selected value
    const [state, setState] = useState<string | null>(null);
    <Autocomplete
      label="State"
      options={states.map(s => ({ label: s.name, value: s.code }))}
      value={state ?? undefined}
      onChange={(val) => setState(val)}
      fullWidth
    />
    ```
  </TabItem>
  <TabItem label="SwiftUI (iOS)">
    ```swift
    import NirmaanTokens

    NavigationView {
        List(filteredStates) { state in
            Text(state.name)
                .onTapGesture { selectedState = state }
        }
        .searchable(
            text: $searchText,
            prompt: "Type to search states…"
        )
        .onChange(of: searchText) { query in
            filteredStates = query.isEmpty
                ? allStates
                : allStates.filter { $0.name.localizedCaseInsensitiveContains(query) }
        }
    }
    .tint(Color(NirColourPrimaryDefault))
    ```
  </TabItem>
  <TabItem label="Compose (Android)">
    ```kotlin
    import dev.nirmaan.tokens.NirmaanTheme

    var query by remember { mutableStateOf("") }
    val filtered = remember(query) {
        states.filter { it.name.contains(query, ignoreCase = true) }
    }

    ExposedDropdownMenuBox(
        expanded = query.isNotEmpty() && filtered.isNotEmpty(),
        onExpandedChange = {}
    ) {
        OutlinedTextField(
            value = query,
            onValueChange = { query = it },
            label = { Text("State") },
            placeholder = { Text("Type to search…") },
            colors = OutlinedTextFieldDefaults.colors(
                focusedBorderColor = NirmaanTheme.colors.borderFocus,
                unfocusedBorderColor = NirmaanTheme.colors.borderDefault,
            ),
            modifier = Modifier
                .menuAnchor()
                .fillMaxWidth()
                .defaultMinSize(minHeight = NirmaanTheme.spacing.touchTarget)
        )
        ExposedDropdownMenu(
            expanded = filtered.isNotEmpty() && query.isNotEmpty(),
            onDismissRequest = { query = "" }
        ) {
            filtered.forEach { state ->
                DropdownMenuItem(
                    text = { Text(state.name) },
                    onClick = { query = state.name }
                )
            }
        }
    }
    ```
  </TabItem>
</Tabs>

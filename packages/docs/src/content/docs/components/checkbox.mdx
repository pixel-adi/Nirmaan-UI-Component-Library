Checkboxes enable multi-select from a list. Unlike Radio buttons, multiple checkboxes can be checked simultaneously. The entire click/tap area is expanded to 44×44px via padding — the visual box remains smaller while the hit area meets WCAG 2.5.8.

## 1. Accessibility Specification

| Attribute | Requirement | Detail |
| :--- | :--- | :--- |
| **Role** | `checkbox` (native `<input type="checkbox">`) | Native element gives keyboard and SR support free. |
| **Label** | `<label>` linked via `htmlFor`/`id` | Clicking label area also toggles the checkbox. |
| **Checked state** | `aria-checked="true/false/mixed"` | `mixed` for indeterminate state (partial group select). |
| **Group context** | Wrapped in `<fieldset>` + `<legend>` | Legend announces the group name to SR users. |
| **Keyboard** | `Space` to toggle | Tab to focus, Space to check/uncheck. |
| **Touch target** | 44×44px minimum via padding | Visual box is 20×20px; padding extends tap area. |
| **Disabled** | `aria-disabled="true"` + `opacity: 0.38` | Stays in tab order — user knows it exists. |
| **Error** | `aria-invalid="true"` + `aria-describedby` | For required checkbox groups (e.g., consent). |

## 4. Component Anatomy

```
[  ✓  ]  Label text
   │
   ├── .nir-e-checkbox__box  → 20×20px visual box
   ├── .nir-e-checkbox__check  → checkmark SVG icon
   └── .nir-e-checkbox__label  → linked <label> element
```

The outer wrapper `.nir-e-checkbox` carries the 44×44px padding-based touch target.

  <TabItem label="SwiftUI (iOS)">
    ```swift
    import NirmaanTokens

    Toggle(isOn: $isChecked) {
        Text("I agree to Terms")
            .font(.custom(NirFontFamilyBody, size: NirFontSizeBodyMd))
            .foregroundColor(Color(NirColourTextDefault))
    }
    .toggleStyle(NirmaanCheckboxStyle(
        activeColor: Color(NirColourPrimaryDefault),
        inactiveColor: Color(NirColourBorderStrong)
    ))
    .frame(minHeight: NirSpacingTouchTarget)
    .accessibilityAddTraits(.isButton)
    ```
  </TabItem>
  <TabItem label="Compose (Android)">
    ```kotlin
    import dev.nirmaan.tokens.NirmaanTheme

    Row(
        verticalAlignment = Alignment.CenterVertically,
        modifier = Modifier
            .defaultMinSize(minHeight = NirmaanTheme.spacing.touchTarget)
            .semantics(mergeDescendants = true) {}
    ) {
        Checkbox(
            checked = isChecked,
            onCheckedChange = { isChecked = it },
            colors = CheckboxDefaults.colors(
                checkedColor = NirmaanTheme.colors.primaryDefault,
                uncheckedColor = NirmaanTheme.colors.borderStrong,
                checkmarkColor = NirmaanTheme.colors.primaryOnPrimary
            )
        )
        Spacer(Modifier.width(NirmaanTheme.spacing.horizontalSmaller))
        Text("I agree to Terms", color = NirmaanTheme.colors.textDefault)
    }
    ```

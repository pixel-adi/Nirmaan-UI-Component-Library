Primitives are the raw values that semantic tokens (`global-variables`) reference. They exist in the `tokens` collection in Figma.

```
Primitive:  tokens/16  →  value: 16
                             ↑
Semantic:   global-variables/spacing/horizontal/base  →  references tokens/16
                             ↑
Component:  .nir-e-button { padding: var(--nir-spacing-horizontal-base) }
```

> [!WARNING]
> **Designers:** You never see or use primitives. They're hidden from the library.
> **Developers:** Never reference `--nir-tokens-*` in CSS. Always use `--nir-spacing-*` or `--nir-colour-*` semantic tokens.

## Why Primitives Exist

<CardGrid>
  <Card title="Scalability" icon="setting">
    Without primitives, every semantic token would contain a hardcoded value. Change `tokens/16` from 16 to 14 and **every** semantic token referencing it updates automatically.
  </Card>
  <Card title="Density Modes" icon="laptop">
    This is how density modes work — by swapping which primitives the semantic tokens reference. Compact mode = smaller primitives. No component code changes needed.
  </Card>
</CardGrid>

## The Raw Palette Architecture

Primitives are prefix-bound by `--nir-` and categorized mathematically:

<CardGrid stagger>
  <Card title="Spacing Grid" icon="setting">
    A rigid **4px multiplier** scaling from `1` (4px) to `16` (64px). Physically prevents arbitrary spacing permutations.
  </Card>
  <Card title="Radius Variants" icon="approve-check-circle">
    Mathematical definitions from `xs` (2px) through `full` (9999px) maintaining precise geometric consistency.
  </Card>
  <Card title="Colour Base" icon="information">
    Complete baseline hex palette (`navy`, `cream`, `red`, `green`) stripped of contextual mappings like 'Success' or 'Danger'.
  </Card>
</CardGrid>

## Numeric Scale Reference

| Token | Value | Used by |
| :--- | :--- | :--- |
| `0` | 0 | `spacing/*/none` |
| `2` | 2 | `corner-radius/smaller`, `stroke/thickness` |
| `4` | 4 | `spacing/*/smallest`, `corner-radius/small` |
| `8` | 8 | `spacing/*/smaller`, `corner-radius/base` |
| `12` | 12 | `spacing/horizontal/small`, `corner-radius/large` |
| `16` | 16 | `spacing/*/base`, `corner-radius/larger` |
| `20` | 20 | `spacing/horizontal/large` |
| `24` | 24 | `spacing/horizontal/larger`, `spacing/vertical/large` |
| `32` | 32 | `spacing/horizontal/largest`, `spacing/vertical/larger` |
| `40–64` | 40–64 | Large layout spacing |
| `l` | 640 | Layout width reference |
| `b` | 360 | Base/mobile width reference |

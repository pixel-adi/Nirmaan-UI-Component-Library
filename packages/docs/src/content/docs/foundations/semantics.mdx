**Tier 2 (Semantic)** tokens map raw primitives to functional meaning. This powers the theming layer—changing Tier 2 values overrides identity securely.

## Theming Combos (Brand × Mode × Density)
The semantic layer resolves on three independent axes natively compiled for Engineering. An intersection must produce a valid accessible output dynamically.

1. **Brand Axis:** (e.g. Base, Central Govt, State Variants) redefining `color.primary`.
2. **Mode Axis:** (Light, Dark, High-Contrast) dictating exact inversions of Surface and Typography contrast bounds (striving for 7:1 AAA ratio constraints in High-Contrast).
3. **Density Axis:** Overriding spatial density to Compact or Spacious mappings mathematically resizing components.

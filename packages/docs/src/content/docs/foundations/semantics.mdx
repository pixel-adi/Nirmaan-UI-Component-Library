**Tier 2 (Semantic)** tokens map raw primitives to functional, contextual meaning. They live in the `global-variables` collection in Figma, and are the only token tier that designers and developers should ever reference directly.

## Theming Combos (Brand Ă— Mode Ă— Density)

The semantic layer resolves on three independent axes, producing a valid accessible output for every combination:

1. **Brand Axis:** Redefines brand expressions (e.g., `colour/primary/base` maps to Central Govt or State identity).
2. **Mode Axis:** Dictates exact colour inversions (Light, Dark, High-Contrast). Strives for 7:1 AAA in High-Contrast.
3. **Density Axis:** Overrides spatial density (Compact, Standard, Spacious) universally across components.

## Cross-Platform Output

Every token generates output for all platforms:

```
Figma token: global-variables / colour / primary / base = #083459

  CSS:      --nir-colour-primary-base: #083459;
  SCSS:     $nir-colour-primary-base: #083459;
  Swift:    static let primaryBase = UIColor(hex: "#083459")
  Android:  <color name="nir_colour_primary_base">#FF083459</color>
  JS:       export const NirColourPrimaryBase = "#083459";
```

## Spacing Tokens

### Horizontal Spacing â€” Inline Direction

Used for `padding-left`, `padding-right`, `column-gap`, and `margin-inline`.

| Token | Value | CSS Variable | When to use |
| :--- | :--- | :--- | :--- |
| `spacing/horizontal/none` | 0 | `--nir-spacing-horizontal-none` | Alignment correction only |
| `spacing/horizontal/smallest` | 4px | `--nir-spacing-horizontal-smallest` | Icon-to-text gap |
| `spacing/horizontal/smaller` | 8px | `--nir-spacing-horizontal-smaller` | Tight inline spacing |
| `spacing/horizontal/small` | 12px | `--nir-spacing-horizontal-small` | Compact button padding |
| `spacing/horizontal/base` | 16px | `--nir-spacing-horizontal-base` | Default button/input padding |
| `spacing/horizontal/large` | 20px | `--nir-spacing-horizontal-large` | Comfortable padding |
| `spacing/horizontal/larger` | 24px | `--nir-spacing-horizontal-larger` | Card inner padding |
| `spacing/horizontal/largest` | 32px | `--nir-spacing-horizontal-largest` | Page margins, section padding |

Nirmaan's theme system works by **overriding semantic tokens**. Components automatically adapt because they reference semantic tokens, not hardcoded values.

```
                Base Theme (Nirman)
                colour/primary/base = #083459
                        │
        ┌───────────────┼───────────────┐
        │               │               │
    Central Govt    Kerala          Tamil Nadu
    #0B3D91         #006B3F         #B71C1C
```

## Creating a Brand Theme

### Step 1 — Define Token Overrides

| Token | Base (Nirman) | Kerala | Tamil Nadu |
| :--- | :--- | :--- | :--- |
| `colour/primary/base` | `#083459` | `#006B3F` | `#B71C1C` |
| `colour/primary/on-base` | `#fcfcfc` | `#fcfcfc` | `#fcfcfc` |
| `colour/primary/stroke` | `#083459` | `#006B3F` | `#B71C1C` |
| `colour/primary/very-light` | `#ced6de` | `#c8e6d0` | `#e6c8c8` |

### Step 2 — Create the Override CSS

```css
[data-brand="kerala"] {
  --nir-colour-primary-base: #006B3F;
  --nir-colour-primary-stroke: #006B3F;
  --nir-colour-primary-very-light: #c8e6d0;
}

[data-brand="tamilnadu"] {
  --nir-colour-primary-base: #B71C1C;
  --nir-colour-primary-stroke: #B71C1C;
  --nir-colour-primary-very-light: #e6c8c8;
}
```

### Step 3 — Apply in Code

## Display Modes

Nirmaan ships with 3 primary display modes:

| Mode | `data-theme` value | Description |
| :--- | :--- | :--- |
| **Light** (default) | `light` or omit | Normal visual scaling. Cream-based surfaces. |
| **Dark** | `dark` | Reversed visual logic targeting deep contrast. |
| **High-Contrast** | `high-contrast` | Aggressive borders, stark WCAG AAA compliance. |

```tsx
// In React, injecting this onto <html> re-writes all CSS variables instantly
document.documentElement.dataset.theme = 'dark';
```

## Theme Scope Rules

> [!CAUTION]
> Status colours are **never customizable**. Red means error everywhere. This is a legal accessibility baseline.

| Token group | Central Govt | State Govt | PSU | Open |
| :--- | :--- | :--- | :--- | :--- |
| `colour/primary/*` | Limited | Yes | Yes | Yes |
| `colour/secondary/*` | Yes | Yes | Yes | Yes |
| `colour/surface/*` | No | Tint only | Yes | Yes |
| `colour/status/*` | **Never** | **Never** | **Never** | **Never** |
| `spacing/*` | No | No | Yes | Yes |
| `corner-radius/*` | No | No | Yes | Yes |

Typography in Nirmaan UI guarantees semantic translations of multi-script configurations mapped perfectly to **Janabhasha (The People’s Language)**. Indian scripts possess entirely different vertical metrics than standard Latin (e.g., Devanagari's Shirorekha).

Therefore, Nirmaan resolves semantic tokens conditionally mapping explicitly to `:lang()` nodes natively, adjusting line-heights on the fly mathematically.

## Multi-Script Metric Adjustments

- **Latin (`:lang(en)`):** 1.50 multiplier.
- **Devanagari (`:lang(hi)`):** 1.70 multiplier (avoids matra collision).
- **Tamil (`:lang(ta)`):** 1.65 multiplier and +0.01em tracking (spacing round forms).
- **Urdu (`:lang(ur)`):** RTL Nastaliq scales heavily to a 1.80 multiplier.

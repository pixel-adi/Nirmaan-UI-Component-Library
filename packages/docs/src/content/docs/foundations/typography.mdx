Typography in Nirmaan is fully token-bound. Every size, weight, line-height, and script multiplier originates in Figma, is transformed by Style Dictionary, and resolves at runtime via CSS custom properties. No hardcoded font sizes exist in component CSS.

## Font Families

| Role | Font | Fallback | Token |
| :--- | :--- | :--- | :--- |
| **Display / Body** | Noto Serif | Georgia, serif | `--nir-font-family-display` |
| **UI / Labels** | Noto Sans | system-ui, sans-serif | `--nir-font-family-body` |
| **Code / Mono** | Noto Sans Mono | Fira Code, monospace | `--nir-font-family-mono` |
| **Devanagari** | Noto Serif Devanagari | Noto Sans Devanagari | via `:lang(hi)` |
| **Tamil** | Noto Serif Tamil | Noto Sans Tamil | via `:lang(ta)` |
| **Urdu / Nastaliq** | Noto Nastaliq Urdu | system | via `:lang(ur)` |

## Multi-Script Metric Adjustments

Different scripts require different typographic multipliers to prevent vertical collision and maintain visual rhythm.

| Script | Lang attribute | Line-height multiplier | Tracking | Reasoning |
| :--- | :--- | :--- | :--- | :--- |
| **Latin** | `:lang(en)` | `1.50` | `0` | Standard, tight default |
| **Devanagari** | `:lang(hi)` | `1.70` | `0` | Matra (vowel diacritic) clearance |
| **Tamil** | `:lang(ta)` | `1.65` | `+0.01em` | Dense round letterforms |
| **Bengali** | `:lang(bn)` | `1.68` | `0` | Matra + ascender clearance |
| **Urdu** | `:lang(ur)` | `1.80` | `0` | Nastaliq vertical depth |

## Token Reference

Full type-scale token names to copy directly into your components:

```css
/* Size scale */
--nir-font-size-display: 3rem;      /* 48px */
--nir-font-size-h1:      2.25rem;   /* 36px */
--nir-font-size-h2:      1.75rem;   /* 28px */
--nir-font-size-h3:      1.375rem;  /* 22px */
--nir-font-size-h4:      1.125rem;  /* 18px */
--nir-font-size-lg:      1.125rem;  /* 18px — body large */
--nir-font-size-base:    1rem;      /* 16px — body default */
--nir-font-size-sm:      0.875rem;  /* 14px — helper / label */
--nir-font-size-xs:      0.75rem;   /* 12px — caption */

/* Weight scale */
--nir-font-weight-display: 700;
--nir-font-weight-heading:  700;
--nir-font-weight-medium:   500;
--nir-font-weight-label:    500;
--nir-font-weight-body:     400;

/* Line-height scale */
--nir-font-line-height-tight:   1.2;
--nir-font-line-height-snug:    1.35;
--nir-font-line-height-normal:  1.5;
--nir-font-line-height-relaxed: 1.75;
--nir-font-line-height-loose:   1.9;
```

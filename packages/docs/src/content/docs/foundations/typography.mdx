Typography in Nirmaan UI guarantees semantic translations of multi-script configurations mapped perfectly to **Janabhasha (The People’s Language)**. Indian scripts possess entirely different vertical metrics than standard Latin (e.g., Devanagari's Shirorekha).

To solve this, Nirmaan resolves semantic typographic tokens conditionally mapping explicitly to `:lang()` nodes natively, adjusting sizes and line-heights on the fly gracefully.

## Multi-Script Metric Adjustments

Different scripts automatically trigger unique architectural multipliers ensuring visual stability.

| Script Context | Multiplier Logic | Reasoning |
| :--- | :--- | :--- |
| **Latin (`:lang(en)`)** | `1.50` | Standard, tight leading default. |
| **Devanagari (`:lang(hi)`)** | `1.70` | Expands leading explicitly to prevent matra (vowel) collisions. |
| **Tamil (`:lang(ta)`)** | `1.65` / `+0.01em` tracking | Expands vertical spacing and slightly tracks out to accommodate dense round letterforms. |
| **Urdu (`:lang(ur)`)** | `1.80` | RTL Nastaliq scales heavily to maintain legibility. |

# Typography

Nirman's typography system handles 12 Indian scripts with script-specific line-heights, letter-spacing, and font families — all driven by design tokens.

## Type Scale

| Token | Size | Use |
|---|---|---|
| `--nir-font-size-display` | 48px | Hero headlines, splash screens |
| `--nir-font-size-heading-xl` | 36px | Page title (h1) |
| `--nir-font-size-heading-lg` | 30px | Section title (h2) |
| `--nir-font-size-heading-md` | 24px | Sub-section (h3) |
| `--nir-font-size-heading-sm` | 20px | Card title (h4) |
| `--nir-font-size-body-lg` | 18px | Lead paragraph |
| `--nir-font-size-body-md` | 16px | Body text (default) |
| `--nir-font-size-body-sm` | 14px | Labels, compact text |
| `--nir-font-size-body-xs` | 12px | Captions, helper text |

## Font Families

The system uses the Noto Sans family — the only typeface family with consistent coverage across all 12 scripts.

| Script | Font | Token |
|---|---|---|
| Latin/English | Noto Sans | `--nir-font-family-latin-sans` |
| Devanagari (Hindi, Marathi) | Noto Sans Devanagari | `--nir-font-family-devanagari-sans` |
| Tamil | Noto Sans Tamil | `--nir-font-family-tamil-sans` |
| Bengali | Noto Sans Bengali | `--nir-font-family-bengali-sans` |
| Telugu | Noto Sans Telugu | `--nir-font-family-telugu-sans` |
| Kannada | Noto Sans Kannada | `--nir-font-family-kannada-sans` |
| Malayalam | Noto Sans Malayalam | `--nir-font-family-malayalam-sans` |
| Gujarati | Noto Sans Gujarati | `--nir-font-family-gujarati-sans` |
| Gurmukhi (Punjabi) | Noto Sans Gurmukhi | `--nir-font-family-gurmukhi-sans` |
| Odia | Noto Sans Oriya | `--nir-font-family-odia-sans` |
| Urdu (RTL) | Noto Nastaliq Urdu | `--nir-font-family-urdu-nastaliq` |
| Monospace | Noto Sans Mono | `--nir-font-family-latin-mono` |

## Per-Script Line Heights

Different scripts need different line heights because of varying glyph heights (matras, shirorekha, conjuncts):

| Script | Line Height | Token | Why |
|---|---|---|---|
| Latin | 1.5 | `--nir-font-line-height-body` | Standard |
| Devanagari | 1.7 | `--nir-font-line-height-body-devanagari` | Shirorekha + matras above/below |
| Bengali | 1.7 | `--nir-font-line-height-body-bengali` | Similar to Devanagari |
| Tamil | 1.625 | `--nir-font-line-height-body-tamil` | Round forms, moderate height |
| Telugu | 1.625 | `--nir-font-line-height-body-telugu` | Wide glyphs |
| Kannada | 1.625 | `--nir-font-line-height-body-kannada` | Moderate height |
| Malayalam | 1.625 | `--nir-font-line-height-body-malayalam` | Dense conjuncts |
| Gujarati | 1.625 | `--nir-font-line-height-body-gujarati` | No headline bar |
| Gurmukhi | 1.625 | `--nir-font-line-height-body-gurmukhi` | Shorter matras than Devanagari |
| Odia | 1.625 | `--nir-font-line-height-body-odia` | Curved forms |
| Urdu | 1.8 | `--nir-font-line-height-body-urdu` | Tallest script (Nastaliq), RTL |

## How It Works

### Automatic script switching

Add `lang` attribute to any HTML element. The typography CSS uses `:lang()` selectors to automatically switch font-family and line-height:

```html
<!-- English (default) -->
<p>Apply for Income Certificate</p>

<!-- Hindi — automatically uses Noto Sans Devanagari, line-height 1.7 -->
<p lang="hi">आय प्रमाण पत्र के लिए आवेदन करें</p>

<!-- Tamil — automatically uses Noto Sans Tamil, line-height 1.625 -->
<p lang="ta">வருமான சான்றிதழுக்கு விண்ணப்பிக்கவும்</p>

<!-- Urdu — automatically uses Noto Nastaliq Urdu, RTL, line-height 1.8 -->
<p lang="ur">انکم سرٹیفکیٹ کے لیے درخواست دیں</p>
```

### In CSS

```css
.my-heading {
  font-family: var(--nir-font-family-heading);
  font-size: var(--nir-font-size-heading-lg);
  font-weight: var(--nir-font-weight-heading);
  line-height: var(--nir-font-line-height-heading);
  letter-spacing: var(--nir-font-letter-spacing-heading);
}
```

### Utility classes

```html
<h1 class="nir-type-h1">Page Title</h1>
<p class="nir-type-body">Body text paragraph.</p>
<span class="nir-type-label">Form Label</span>
<span class="nir-type-caption">Helper text</span>
<code class="nir-type-mono">APP-2026-00451</code>
```

## Font Weights

| Token | Weight | Use |
|---|---|---|
| `--nir-font-weight-body` (400) | Regular | Body text, descriptions |
| `--nir-font-weight-label` (500) | Medium | Labels, button text, nav items |
| `--nir-font-weight-heading` (600) | Semibold | Headings, card titles |
| `--nir-font-weight-emphasis` (700) | Bold | Display text, emphasis |

## Loading Fonts

Add this to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&family=Noto+Sans+Devanagari:wght@400;500;600;700&family=Noto+Sans+Tamil:wght@400;500;600;700&family=Noto+Sans+Bengali:wght@400;500;600;700&display=swap" rel="stylesheet">
```

Load only the scripts you need. Each script adds ~50–80KB.
